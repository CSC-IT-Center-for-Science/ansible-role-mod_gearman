ansible-role-mod_gearman
=========

Ansible role to install and configure mod_gearman.

Dependencies
------------

The epel repo must be available.

Example Playbook
----------------

* You can simply use this role like
```
- hosts: servers
  roles:
     - { role: ansible-role-mod_gearman }
```
